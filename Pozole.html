Frontend

Frontend es la parte de un programa o dispositivo a la que un usuario puede acceder directamente. Son todas las tecnologías de diseño y desarrollo web que corren en el navegador y que se encargan de la interactividad con los usuarios.

Temas tratados

  • Configuración básica y conocimientos sobre cómo aprender
  • Estándares web y mejores prácticas
  • HTML, el lenguaje que da estructura y significado al contenido web
  • CSS, el lenguaje utilizado para aplicar estilo a las páginas web
  • JavaScript
  • Herramientas que se utilizan para facilitar el desarrollo web

Prerrequisitos No necesitas conocimientos previos para empezar este curso. Todo lo que necesitas es una computadora que pueda ejecutar navegadores web modernos, una conexión a Internet y la voluntad de aprender. Si no estás seguro de si el desarrollo de la interfaz de usuario web es para ti, y/o deseas una breve introducción antes de comenzar un curso más largo y completo, consulta una Introducción a la Web.
